***Commands used while creating simple CRUD based application using it. 
______________________________________________________________________________________________________ 

go mod init new-module  :: To initialize the new project 

go get -u github.com/gorilla/mux :: installing new framework named mux into our app 

go get -u gorm.io/gorm :: installing the orchestration tool  

go get -u gorm.io/driver/postgres :: installing the connection driver for postgreSQL   

go build :: Building the modules which is used within our app 

.\new-module.exe :: executing the build module or app 

go get github.com/sirupsen/logrus :: for log
